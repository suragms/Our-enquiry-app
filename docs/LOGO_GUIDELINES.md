# HexaStack Logo Guidelines

## Primary Logo

- **Display:** "HEXASTACK" only (no "AI Solutions")
- **Style:** Minimal, flat, vector. Inspired by Stripe, Notion, Linear, Vercel.
- **Format:** SVG preferred for web; PNG for email/social where needed.

## Logo Variants

| Variant | Use case |
|---------|----------|
| **Primary (light)** | Light background (#F5F5F5) — dark mark |
| **Primary (dark)** | Dark background (#0D0D0D) — light mark |
| **Icon only** | Favicon, app icon, small spaces |
| **Favicon** | 32×32 (or SVG) for browser tab |
| **App icon** | 512×512 for PWA / app stores |

## Typography (Wordmark)

- **Recommended:** Sans-serif, geometric. Options: Inter (current), Plus Jakarta Sans, or similar. Weight: 600–700 for "HEXASTACK."
- **Spacing:** Letter-spacing slightly tight (e.g. -0.02em) for wordmark.
- **Pairing:** Same sans for UI; optional display font for hero headlines only.

## Clear Space & Sizing

- **Clear space:** Minimum space around logo = height of "H" (or icon height) on both sides.
- **Minimum size:** Wordmark height no smaller than 24px on screen; icon no smaller than 16px.

## Brand Colors

| Role | Hex | Use |
|------|-----|-----|
| **Background (dark)** | #0D0D0D | Primary page background |
| **Background (light)** | #F5F5F5 | Sections, cards on dark |
| **Foreground (on dark)** | #F5F5F5 | Text, logo light |
| **Foreground (on light)** | #0D0D0D | Text, logo dark |
| **Accent (electric blue)** | Gradient | CTAs, links, highlights |
| **Accent start** | #2563EB | Blue |
| **Accent end** | #06B6D4 | Cyan |

Electric blue gradient: `linear-gradient(135deg, #2563EB 0%, #06B6D4 100%)` (or similar).

## Usage Rules

- Do not stretch, rotate, or add effects to the logo.
- Do not place on busy backgrounds without sufficient contrast.
- Do not pair with competing wordmarks in the same lockup.
- Footer and legal: use "HexaStack Solutions" as text; logo can remain "HEXASTACK."

## File Naming

- `logo.svg` — default (dark mark for light bg)
- `logo-dark.svg` — light mark for dark bg
- `favicon.svg` / `favicon.ico` — icon only
- `apple-touch-icon.png` — 180×180 for iOS
